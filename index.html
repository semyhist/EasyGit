Connect GitHub 🔗

We need a Personal Access Token to read and update your repositories on your behalf.

1
Go to github.com/settings/tokens/new
2
Give it a name like "EasyGit"
3
Set expiration: No expiration (or 1 year)
4
Select scopes below, then click Generate token
✅ repo — read/write repositories (topics, description, README)
✅ user — read your profile information
👁️
Your token is stored locally on your device. It is never sent to any server except GitHub's API.
